1. Introduction
Welcome to the Library Application README! In this document, there are instructions on how to compile, run, and to use the Library Application. The Library Application helps with sorting call numbers in numerical and then alphabetical order. 

2. Project Structure
The project consists of one main components:
- **LibraryApp**: C# software application.


3. Requirements 
Before you begin, these are the requirements that must be installed:
- Visual Studio (Recommended: Visual Studio 2019 or later) with C#.
- .NET Framework (Target Framework: .NET Framework 4.7.2).

4. Compilation and Execution
Follow these steps to compile and run the Time Management Application:

1. Open the solution file `LibrarytApp.sln` in Visual Studio.

2. Build the solution by clicking `Build` in the menu.

3. Set the `LibraryApp` project as the startup project:
   - Right-click on `LibraryApp` in the Solution Explorer.
   - Select "Set as Startup Project."

4. Click the "Start Debugging" button to run the application.

5. Usage
Once the application is running, you can perform the following tasks:
- Replacing books.
- Generate a sequence of numbers.
- Sort them in numerical and the alphabetic order.
- Start the and get your points.

6. Additional Notes
- This application does not persist user data between runs. Data is stored in memory while the application is running.


Thank you for using the Library Application! If you encounter any issues or have any questions, please refer to the project's documentation or contact the developer.
